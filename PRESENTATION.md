# SafeRoute G - 발표 자료
## 배달 라이더를 위한 열 스트레스 안전 경로 시스템

---

## 🎯 프로젝트 개요

**SafeRoute G**는 폭염 환경에서 일하는 배달 라이더의 **열 스트레스를 최소화**하는 경로 추천 시스템입니다.

### 데모 URL
- **웹사이트**: https://vibe-orpin-psi.vercel.app
- **경로 비교**: https://vibe-orpin-psi.vercel.app/route
- **백엔드 API**: https://saferoute-g-backend.onrender.com

---

## 🔥 문제 정의

### 배달 라이더의 열 스트레스 위험

1. **장시간 야외 노출**
   - 하루 6-10시간 야외에서 배달 업무 수행
   - 직접적인 태양 노출

2. **폭염 취약**
   - 오토바이/자전거로 에어컨 없이 이동
   - 고온 환경에서 신체 부담 증가

3. **경로 선택권 없음**
   - 기존 네비게이션은 **열 환경 고려 없이** 최단 경로만 제공
   - 빠른 도로 = 넓은 도로 = **그늘 없음** = 열 노출 극대화

| 결과 | **열 노출 극대화** 🔥 | **열 노출 30% 감소** ❄️ |

## 🛠️ 기술 스택

### Frontend (Next.js 15.3.6)
- **React 18** + **TypeScript**
- **Tailwind CSS** - UI 스타일링
- **Leaflet** - 지도 시각화
- **Vercel** - 배포

### Backend (FastAPI + Python 3.11)
- **FastAPI** - REST API 서버
- **OSMnx** - OpenStreetMap 도로 네트워크
- **NetworkX** - Modified A* 알고리즘
- **GeoPandas** - 지리 데이터 처리
- **Render** - 배포

### 데이터 소스
- **경기도 기후데이터 플랫폼 API**
  - WFS 서비스 (녹지, 쉼터 위치)
- **OpenStreetMap**
  - 도로 네트워크 (수원시)

## 🔬 핵심 알고리즘

### Modified A* 다중목표 경로 최적화

**비용 함수**:
```
TotalCost = α·Time + β·HeatStress + γ·SolarExposure
          - δ·GreenBonus - ε·ShadeBonus
```

**모드별 가중치**:

| 모드 | Time (α) | Heat (β) | Solar (γ) | Green (δ) | Shade (ε) |
|------|----------|----------|-----------|-----------|-----------|
| 🚀 Fast | **10.0** | 0.1 | 0.1 | 0.0 | 0.0 |
| ⚖️ Balanced | 1.0 | 1.5 | 1.0 | 1.0 | 1.0 |
| 🌳 Heat-Safe | 0.1 | **5.0** | **4.0** | **3.0** | **3.0** |

### 도로 타입별 열 환경

| 도로 타입 | UTCI | 그늘 | 설명 |
|----------|------|------|------|
| 고속도로/간선도로 | **36°C** | 10% | 가장 뜨거움 🔥 |
| 주요 도로 | 34°C | 20% | 뜨거움 |
| 중간 도로 | 32°C | 30% | 보통 |
| 주거지역 도로 | **30°C** | 50% | 시원함 ❄️ |
| 이면도로 | **29°C** | 60% | 가장 시원함 ❄️ |
